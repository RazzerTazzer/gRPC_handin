To run the program first run the command: 
dotnet run server/server.go 

Afterwards run the the command where port is the wished port, and name is the clients name:
dotnet run client/client.go -port xxxx -name xx

To join write from the client “join” in the command line

To leave write “leave” in the command line

To send message write any string in the command line and hit enter after joining.
